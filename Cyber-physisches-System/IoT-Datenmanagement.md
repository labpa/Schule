## DAS - Direct Attached Storage 
DAS-Datenträger sind dem Server physisch angefügt
+ Vorteile
    + Einfach zu konfigurieren
    + Preisgünstige Lösung

+ Nachteile
    + Isoliert, da nur einem einzelnen Server anfügt
    + Langsam
